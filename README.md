# zn-plugin-admin

前端基于zeanium-react-web框架, 后端基于zeanium-node框架的后台管理平台模块, 包含基础的数据管理，菜单管理，用户管理，角色管理，权限管理，系统参数管理等模块，直接安装即可使用。后期持续完善中...
